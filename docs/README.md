# 숫자 야구 게임

## 기능 구현 목록

---
### 1.난수 생성

- 상태방(컴퓨터)의 숫자를 랜덤으로 생성한다
- camp.nextstep.edu,missionutils를 사용하여 구현한다.
- 중복 숫자가 아니여야한다.

### 2. 사용자의 숫자 입력

- 숫자가 아닌 경우
- 입력값의 길이가 3개 이상인 경우
- 서로 같은 수가 있을 경우


### 3. 숫자맞추기

- 스트라이크: 같은 수, 같은 자리
- 볼: 같은 수, 다른 자리
- 낫싱: 하나도 없는 경우
- 받아온 컴퓨터와 사용자의 리스트 인덱싱을 통해서 비교하여 결과 출력

### 4. 게임 재실행 여부

- 게임의 재진행 여부를 확인하기
- 재시작: 1
- 종료: 2

### 5. 실행

- 전체적인 메서드를 순서대로 조합하여 실행할 수 있도록 함
